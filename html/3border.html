border : 테두리 선을 지정하는 속성



border: 두께(width) 종류(style) 색상(color) 순으로 입력





기본 선 색상으로 black이 설정되어있음



border-width,border-style, border-color 등으로 한가지만 설정 가능



border-top-width 등으로 한쪽면의 설정만 가능



style 종류





선 없음(none) * 기본값



실선(solid) * 자주사용



점선(dotted)



파선(dashed) * 자주사용



두줄선(double)



홈이 파여있는 모양(groove)



솟은 모양(ridge, groove의 반대)



요소 전체가 들어간 모양(inset)



요소 전체가 나온 모양(outset)



color - 기본색(검정), transparent(투명)



색상 표현



1. 색상 이름 - 브라우저에서 제공하는 색상 이름 (ex. red, yellow, tomato등)



2. Hex 색상코드 - 16진수 색상 * 많이 사용 (ex. #000, #ffffff등)



3. RGB - 빛의 삼원색 (ex. rgb(255,255,255)등)



4. RGBA - 빛의 삼원색 + 투명도 (ex. rgba(0,0,0,0.5)-투명도는 0~1까지의 숫자로 표현



5. HSL - 색상, 채도, 명도 (ex. 120, 100%, 50%)



6. HSLA - 색상, 채도, 명도 + 투명도 (ex. 120, 100%, 50%, 0.3)
