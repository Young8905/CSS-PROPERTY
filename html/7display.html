display 요소의 화면 출력(보여짐)특성



1. block 상자(레이아웃) 요소 (ex. div등), 각 요소에 이미 지정되어 있는 값


가로, 세로 값을 지정 할 수 있음



2. inline 글자 요소 (ex. span등), 각 요소에 이미 지정되어 있는 값


가로, 세로 값을 지정해도 설정되지 않음



3. inline-block 글자+상자 요소, 각 요소에 이미 지정되어 있는 값



4. flex 플렉스 박스 (1차원 레이아웃), 따로 지정해서 사용하는 값



5. grid 그리드 (2차원 레이아웃), 따로 지정해서 사용하는 값



6. none 보여짐 특성 없음, 화면에서 사라짐, 따로 지정해서 사용하는 값



7. 기타 table, table-row, table-cell 등
